# Dynamic Metal Price Calculator

**By Aolo Software**

Dynamic Metal Price Calculator allows WooCommerce store owners to set and manage gold, silver, and custom metal rates from a simple admin interface. It calculates product prices dynamically based on metal type, weight, base price, wastage, making charges, markup, GST, and shipping. Supports variable products with per-variation pricing.

---

## Features

- Set gold, silver, and custom metal rates from admin
- Supports 24K, 22K, 20K, and 18K gold
- Per-product settings:
  - Weight
  - Base price
  - Wastage
  - Making charge
  - Markup
- Global and per-product GST support
- Global and per-product flat shipping
- Variable product support â€“ configure metal pricing for each variation
- Custom metal formulas
- Dynamic price calculation at cart and checkout

---

## Installation

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure metal rates and product settings in the plugin admin page.

---

## FAQ

**Q: Does this plugin support multiple metals?**  
A: Yes, you can configure gold, silver, and custom metals.

**Q: How does GST work?**  
A: GST can be applied globally or per product. The plugin calculates GST dynamically during price computation.

**Q: Does this plugin support variable products?**  
A: Yes, you can set metal type, weight, base price, wastage, making charge, markup, GST, and shipping for each variation individually.
